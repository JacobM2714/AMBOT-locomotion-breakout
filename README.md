# AMBOT-locomotion-breakout
PCB that supplies power to prototype. Will be modified in future to connect power, bluetooth, drivers, and STM microcontroller modules

All third-party ECAD Files are found in the 'ECAD-Addon-Files' Folder. After importing the project to KiCAD add the symbols and footprints in their respective management tool in: Prefrences -> Manage Symbol Libraries or Manage Footprint Libraries
